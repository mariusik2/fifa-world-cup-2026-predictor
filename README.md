# 🏆 FIFA World Cup 2026 Predictor

A professional Google Sheets prediction pool for the FIFA World Cup 2026.

## Features

* ⚽ Official FIFA World Cup 2026 fixtures
* 🔄 Automatic results update using API-Football
* 👥 Multiple players
* 🥇 Live standings
* 📊 Statistics dashboard
* 🎯 Exact score prediction scoring
* 🕒 Automatic hourly updates
* 📱 Mobile friendly

---

## Scoring Rules

| Prediction              | Points |
| ----------------------- | -----: |
| Exact score             |      3 |
| Correct goal difference |      2 |
| Correct winner/draw     |      1 |
| Wrong prediction        |      0 |
